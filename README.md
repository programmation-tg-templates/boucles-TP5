# Boucles - TP5 - La factorielle

## Consignes

Créer une fonction `factorielle(n: number): number` qui renvoie la factorielle d’un nombre entré en paramètre.

## Rappel mathématique

La factorielle d'un nombre entier naturel `n` est le produit des nombres entiers strictement positifs inférieurs ou égaux à `n`.

Par convention, la factorielle de 0 est 1. 

## Exemples

- `factorielle(1)` => $1! = 1$
- `factorielle(2)` => $2! = 2 \times 1 = 2$
- `factorielle(3)` => $3! = 3 \times 2 \times 1 = 6$
- `factorielle(4)` => $4! = 4 \times 3 \times 2 \times 1 = 6$
- `factorielle(5)` => $5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$
